> hi, i'm LoneWriter!

i'm the webmaster of this site... or maybe the 'discoverer of the void' but idk if that counts.. i don't know how i discovered neocities.. but i've enjoyed making this site and improving it over time :D

i love writing and obsessing over my projects and my OCs!, i also like programming and hope it's something i can do in the future.. although i wouldn't say i'm the greatest at it yet, probably quite the opposite tbh :p you'll be able to find some of the projects i've worked on here :D gaming is another passion of mine, you'll be able to find a list of all the games i've enjoyed playing below.. there's a lot ;-;

i have quite a few other passions and obsessions, but i'll save those for shrines or a list for the below section on this page :)

in terms of my personality, i'm an INFJ, i'm quite introverted irl and online, but if i know someone a little further, then i'll open up. this website is where i can kinda be myself, whilst kinda worrying about all the crap that goes on, on the internet..

tbh.. i'm not very good at this whole introductions thing.. whatever, it doesn't matter, you'll soon forget i exist ^^

> username... (i mean... why am i called LoneWriter?)

well... i originally went by an old alias which was generated by Xbox when i made my account... but that was 2017 and i decided to change it in 2025. like all good usernames, it was randomly thought of one night and it kinda just stuck.

the 'Lone' part originates from the fact that i'm kinda socially anxious and don't really have many friends that i keep in contact with which was especially the case when high school ended and now i barely speak to any of the 3 (maybe 5?) friends i made there. I guess as time moved on, i never really made any more friends for a few years... so that part of the username kinda just made itself... although i am trying to make some more friends now ^^

as for the 'Writer' part, i was really getting into writing at the time and had recently finished my fourth draft of One Eye, One Girl. i really enjoyed writing and so it kinda had to be.

despite me changing my username, when i hop on a game with an old friend, which i somehow keep in contact with, we sometimes wonder why i changed it and they kinda want me to change it back to what it was, if they read this, then they'll know :)

> about the website:

this website has been through 3 revisions in less than a year somehow..?, and every time i've gone to change it, i've loved doing it... unless there's some silly css error that i'm too blind to notice, this has been my main way of learning html, css and a little bit of javascript - hopefully it's something i'll be able to do in the future a little more.

as for the current layout of version 3 of this website, it's kinda heavily inspired by Sadgrl's layout builder with a few more aspects of the css hand coded... cuz any other layout i try... i'm not a huge fan of.

as for the previous versions, which you may have seen or not... idk, the first version of the this website orginally began through a template i had downloaded here (it was the Nil layout) and i then discovered that i could upload it to neocities, so here we are today :) and as for the second layout, it used Sadgrl's layout builder with some of the parameters changed... obviously..

as a final thing, i'll just say the majority of the code on this website is held together with some duct tape and a motivation to learn web dev... please don't copy any of the code because i can guarantee you'll find better examples of what i've done elsewhere :)

also... you might be wondering why it's called 'the void' well... i made this website alongside a game i was making... in which the main location is called the void... so there you go :)
